Учебные упражнения

На данном уроке вы отработаете создание классов, добавление полей и методов.

Задания:

  1. Создайте класс с названием Car.
  2. У класса Car должны быть приватные поля brand, model, yearOfEdition, mileage.
  3. Класс должен содержать конструктор без параметров и конструктор с параметрами для всех полей.
  4. В классе должны быть определены сеттеры и геттеры для всех полей.
